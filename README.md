## Hospital Management System

A Windows Forms application built in C# for managing hospital operations such as patient records, health monitoring, and basic medical assessments. The system provides a user-friendly interface and modular structure, making it easy to maintain and extend.



## Features

• User authentication (login/logout)
• Dashboard with patient and health statistics
• Add and manage patient records
• BMI (Body Mass Index) checker
• Heart rate monitor
• Blood pressure checker
• Sleep quality checker
• Hydration checker
• Visual navigation with highlighted active sections
• Confirmation dialogs for critical actions



## Technologies Used

• C#
• .NET Framework (WinForms)
• Visual Studio


## Getting Started

1. Clone the repository:
    https://github.com/Kalai20031/Hospital_Management_System_1st_Sem.git
2. Open the solution:
   • Open the `.sln` file in Visual Studio

3. Build and run:
   • Press **F5** or click **Start** in Visual Studio


## Project Structure

• MainForm.cs – Main navigation and UI logic
• LoginForm.cs – User authentication
• Dashboard.cs – Overview of patient data and statistics
• AddPatient.cs – Add and manage patient records
• BMIChecker.cs – Calculate patient BMI
• HeartRateMonitor.cs – Monitor heart rate
• BloodPressureChecker.cs – Check blood pressure levels
• SleepQualityChecker.cs – Analyze sleep patterns
• HydrationChecker.cs – Track hydration levels
